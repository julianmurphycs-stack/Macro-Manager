Description: This is a desktop macro automation tool that allows users to create, edit, save, and run macros through a simple GUI. Macros can be run a specified number of times or looped infinitely. They are capable of automating keyboard inputs, mouse clicks, mouse dragging, and timed waiting periods.

How to use:

Navigate to the directory that contains your MacroApp.java: 
cd (path to MacroApp)

Compile the program: 
javac MacroApp.java

Run the program: 
java MacroApp
